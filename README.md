# Image Slider 

A responsive, mobile-friendly image slider with auto-play functionality, smooth transitions, and interactive controls.

## Features

- **Auto-play functionality** with configurable interval
- **Manual navigation** with previous/next buttons
- **Slide indicators (dots)** for direct slide selection
- **Smooth fade transitions** between slides
- **Mobile-friendly** with touch/swipe support
- **Responsive design** that works on all screen sizes
- **Play/Pause and Stop controls** for auto-play
- **Clean, modern UI** with subtle animations
